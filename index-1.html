About Me

I am currently pursuing a Master of Science in Big Data Science at Queen Mary University of London, where I am building a strong foundation in advanced data analytics and large-scale data processing. My coursework includes Data Mining, Machine Learning, Natural Language Processing, Neural Networks, and Big Data Processing, through which I have gained practical experience in developing predictive models, working with real-world datasets, and utilising tools such as Python, Spark, and TensorFlow.

During this time, I interned at two textile industries, where I worked with production and quality data to identify inefficiencies and propose data-driven solutions to real-world operational challenges.

With a blend of engineering expertise and advanced data science training, I am well-positioned to bridge the gap between domain knowledge and data-driven insight. I am eager to apply my analytical skills and technical proficiency to uncover meaningful patterns, support strategic decision-making, and drive measurable impact as a data analyst.

Projects

Fabric Defect Detection

This project focuses on automating fabric defect detection using Deep Learning models. I built a pipeline combining a custom CNN-based binary classifier and YOLOv8 for defect localisation, trained on the TILDA textile dataset. The system accurately classifies fabrics as defective or clean and highlights defect regions with bounding boxes. The final solution was deployed using a Streamlit app for real-time inference, enabling industry-ready quality inspection.

Demo GitHub

CIFAR-10 Classification

A PyTorch image classifier for CIFAR-10 that uses attention over multiple kernel sizes inside each block, plus BatchNorm, label smoothing, dropout, and a CosineAnnealing LR schedule. The model learns to soft-select among 1×1, 3×3, 5×5, and 7×7 convolutions per block.

GitHub

Apache Spark for Big Data Analysis

This project demonstrates the use of Apache Spark (PySpark) for analysing and processing large-scale datasets across multiple domains, including Twitter data, movie ratings, Chicago taxi trips, and HDFS log streaming. Each task applies big data techniques, from ETL and aggregation to graph analytics and real-time streaming, to derive meaningful insights from raw data stored in AWS S3.
